8. 衝堂



  請檢查輸入的三門課程是否衝堂。
    依序輸入 :

      課程編號 (科目名稱 + 4位數字)、

      上課小時數 (1-3 小時)、

      上課時間 (依小時數輸入上課時間, 星期 1-5, 第 1-9,a,b,c 節)。

輸入輸出說明


  輸入說明
  輸入三門課程的資訊
  每一門課程的資訊
    第一行 : 輸入課程編號(字串)
    第二行 : 上課小時數 H (整數，1<=H<=3)

    接下來有 H 行

    每一行為一個字串，第一個字元表示星期幾，第二個字元表示第幾節

  輸出說明
    輸入任何錯誤，輸出 -1
    若無發生衝堂，則輸出correct

    若發生衝堂 :
      輸出所有衝堂的課程，每次輸出兩個衝堂的課程編號，
    以及在哪一天的哪一節衝突，參考下列格式 :
{課程1編號},{課程2編號},{衝堂在哪天哪節}先輸入的課程為課程1，後輸入的課程為課程2

輸入範例解釋


  Chinese1001 (第一門課課程編號)
  3 (3小時，每節課1小時)
  11 (星期1 第1節課)
  12 (星期1 第2節課)
  13 (星期1 第3節課)
  English1002 (第二門課課程編號)
  3 (3小時，每節課1小時)
  21 (星期2 第1節課)
  22 (星期2 第2節課)
  23 (星期2 第3節課)
  Math1003 (第三門課課程編號)
  3 (3小時，每節課1小時)
  31 (星期3 第1節課)
  32 (星期3 第2節課)
  13 (星期1 第3節課)

輸出範例解釋


  Chinese1001,Math1003,13
  (課程Chinese1001跟課程Math1003，在星期1第3節衝堂，因課程Chinese1001先輸入，所以課程Chinese1001放前面)

Tips



  使用List(列表、序列)儲存資訊
  List為Python基本的數據結構，List中每個元素都分配一個數字、他的位置或索引。
  第一個索引是0，第二個索引是1，以此類推
  要宣告list，使用中括號[]，如下 

                            list = ['physics', 'chemistry']
                            list[0] = 'physics'
                            list[1] = 'chemistry'
  要往後新增元素的話，使用append

                          list.append('Programming')
                          list = ['physics', 'chemistry', 'Programming']

                        # 判斷課程輸入是否符合規格
                        def check(k):
                            # 檢查課程代碼的第一個字元是否在 '1' 到 '5' 之間
                            if '1' <= k[0] <= '5':
                                # 檢查第二個字元是否在 '1' 到 '9' 之間
                                if '1' <= k[1] <= '9':
                                    return True
                                # 或者第二個字元是否為 'a', 'b', 或 'c'
                                elif k[1] in ('a', 'b', 'c'):
                                    return True
                                else:
                                    return False
                            else:
                                return False
                        
                        # 輸入N門課程的時間資訊
                        def input_times(n):
                            times_list = []  # 儲存課程時間的列表
                            flag = True      # 標記課程是否衝堂
                            for _ in range(n):
                                k = input()           # 輸入一個時間
                                if check(k):
                                    times_list.append(k)  # 如果輸入正確，加入列表
                                else:
                                    flag = False          # 輸入不符合規格，標記為False
                            return times_list, flag       # 返回時間列表和標記
                        
                        # 判斷兩門課程是否有衝堂
                        def Conflict(courseID1, list1, courseID2, list2):
                            flag = True  # 初始假設沒有衝突
                        
                            # 檢查list1中的每個時間是否出現在list2中
                            for time in list1:
                                if time in list2:
                                    flag = False
                                    print('{}, {}, {}'.format(courseID1, courseID2, time))
                            return flag
                        
                        def func():
                            # 輸入第一門課程的ID和時間數量
                            course_a_id = input()
                            course_a_length = int(input())
                            course_a_time, flag1 = input_times(course_a_length)
                        
                            # 輸入第二門課程的ID和時間數量
                            course_b_id = input()
                            course_b_length = int(input())
                            course_b_time, flag2 = input_times(course_b_length)
                        
                            # 輸入第三門課程的ID和時間數量
                            course_c_id = input()
                            course_c_length = int(input())
                            course_c_time, flag3 = input_times(course_c_length)
                            
                            # 判斷所有課程的輸入是否都符合規格
                            Flag = flag1 and flag2 and flag3
                        
                            if not Flag:
                                print("-1")  # 如果有任何輸入不符合規格，輸出 -1
                            else:
                                # 判斷三門課程之間是否有衝堂
                                isCorrect1 = Conflict(course_a_id, course_a_time, course_b_id, course_b_time)
                                isCorrect2 = Conflict(course_a_id, course_a_time, course_c_id, course_c_time)
                                isCorrect3 = Conflict(course_b_id, course_b_time, course_c_id, course_c_time)
                        
                                # 如果沒有任何衝堂，輸出 "correct"
                                isCorrect = isCorrect1 and isCorrect2 and isCorrect3
                                if isCorrect:
                                    print("correct")
                        
                        # 主程式入口
                        if __name__ == '__main__':
                            func()
